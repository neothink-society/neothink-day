## Creating New Templates

1. Start with reference template: `../original-emails/level3-welcome-recording.html`
2. Reference design system: `../EMAIL-DESIGN-SYSTEM.md`
3. Reference content guidelines: `../EMAIL-CONTENT-GUIDELINES.md`
4. Follow quick start guide: `../QUICK-START-GUIDE.md`
5. Save in appropriate audience directory
6. Test thoroughly before sending

## Naming Conventions

Use descriptive, hyphenated lowercase names:
- `workshop-announcement-[date].html`
- `vip-session-countdown-[date].html`
- `level3-welcome-recording.html`
- `registration-confirmation-[month-year].html`
